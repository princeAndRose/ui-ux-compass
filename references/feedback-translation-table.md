# Feedback Translation Table

Convert vague feedback into executable constraints. Ask for evidence only when the surface or task is unclear.

| User wording | Likely meaning | Ask or inspect | Patch direction |
| --- | --- | --- | --- |
| Ugly | weak visual focus, mixed hierarchy, inconsistent rhythm | inspect first viewport and P0 | strengthen type scale, spacing, grouping, and CTA priority |
| Too cramped | overloaded viewport, dense controls without grouping | inspect density and section spacing | increase grouping whitespace, collapse secondary controls |
| Too empty | weak content model or low-value whitespace | inspect real data and states | add useful metadata, state content, or stronger anchors |
| Too busy | too many accents, borders, CTAs, or sections | inspect competing visual variables | reduce accents, merge sections, expose progressive detail |
| Generic / template-like | layout not shaped around task | compare against surface playbook | replace equal cards with task-specific archetype |
| Not premium | weak typography, uncontrolled effects, cheap decoration | inspect type, color, elevation | simplify effects, refine type and color hierarchy |
| Not product-like | missing real data or states | inspect loading/empty/error/selected/disabled | add realistic state coverage and domain copy |
| Feels off / weird | alignment, spacing, contrast, or archetype mismatch | inspect screenshots and component boundaries | fix rhythm, alignment, state feedback, or layout model |
| 不好看 / 太丑 | visual focus is unclear or hierarchy is flat | inspect P0 and first scan | strengthen focus, type, spacing, and grouping |
| 太挤 / 太紧凑 | density exceeds task tolerance | inspect viewport load and control grouping | reduce equal-weight content, improve spacing rhythm |
| 太空 / 太散 | content feels under-modeled | inspect actual data and empty states | add anchors, metadata, state copy, or better structure |
| 太乱 / 太花 | visual variables compete | inspect accents, borders, CTA count | reduce decoration, consolidate sections, clarify priorities |
| 模板感 / 套模板 | generic card grid or stock SaaS layout | inspect archetype fit | choose dashboard/workspace/list-detail/form archetype |
| 不高级 / 廉价 | typography/elevation/color lacks restraint | inspect visual vocabulary axes | simplify effects, improve type and color calibration |
| 不像产品 / 像 demo | lacks realistic states and data behavior | inspect state checklist | add real data density, affordances, and recovery states |
| 没重点 / 信息层级不清 | P0/P1/P2 hierarchy missing | inspect scan path | define first focus, main CTA, and priority tiers |
| 有点怪 / 看起来怪 | mismatch or small craft breaks trust | inspect alignment, contrast, layout fit | fix alignment, rhythm, responsive behavior, state completeness |
