# UI Anti-patterns

- Equal-weight card soup
- Generic SaaS dashboard
- Random gradient hero
- Heavy shadow everywhere
- Meaningless illustration
- Decorative glassmorphism
- CTA overload
- Data with no hierarchy
- Form fields without grouping
- Modal for everything
- Empty state as afterthought
- Desktop-only layout
- Landing page that hides the product
- Metrics with no decision attached
- Settings page without grouping or save feedback
- Editor without dirty, error, preview, or disabled states
