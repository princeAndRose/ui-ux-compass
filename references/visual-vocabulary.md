# Visual Vocabulary

Use these axes to turn vague taste feedback into executable design constraints.

## Axes

- Information density: `low`, `medium`, `high`
- Structure: `loose`, `moderate`, `strong-grid`
- Decoration: `restrained`, `balanced`, `expressive`
- Product feel: `tool`, `content`, `brand`, `community`, `data`, `editor`
- Mood: `calm`, `focused`, `energetic`, `warm`, `technical`, `premium`
- Hierarchy method: `typography`, `spacing`, `contrast`, `color-block`, `shadow`, `motion`
- Commercial feel: `low`, `medium`, `high`
- Product realism: `demo-like`, `prototype-like`, `production-like`

## Translation Table

- "Ugly": unclear visual focus, flat hierarchy, mixed variables, weak layout rhythm.
- "Too cramped": insufficient section spacing, overloaded first viewport, weak grouping.
- "Not premium": uncontrolled shadows, cheap decoration, weak typography, color imbalance.
- "Template-like": generic card grid, no task-shaped layout, equal-weight sections.
- "Not product-like": missing loading, empty, error, selected, disabled, or real-data states.
- "Too busy": competing CTAs, too many borders, too many accents, weak progressive disclosure.
- "Too empty": weak content model, no states, insufficient affordances, excessive whitespace around low-value content.

## Chinese Feedback Translation Table

- "不好看" / "太丑": unclear first visual focus, flat hierarchy, weak type scale, mixed spacing rhythm.
- "太挤" / "太紧凑": overloaded viewport, insufficient grouping, weak whitespace hierarchy, too many equal-weight controls.
- "太空" / "太散": sparse content model, weak affordances, low-value whitespace, missing section anchors.
- "太乱" / "太花": competing accents, noisy borders, unclear action priority, too many unrelated visual treatments.
- "模板感" / "套模板": generic card grid, no task-shaped layout, repeated equal-weight modules, weak product-specific structure.
- "不高级" / "廉价" / "塑料感": uncontrolled shadows, weak typography, color imbalance, decorative effects without functional hierarchy.
- "不像产品" / "像 demo" / "不像真实产品": missing real-data density, empty/loading/error/selected/disabled states, unrealistic copy or affordances.
- "没重点" / "信息层级不清" / "视觉层级不清": missing P0/P1/P2 hierarchy, weak primary action, unclear scan path.
- "看起来怪" / "有点怪": inspect alignment, spacing rhythm, contrast, state completeness, and whether the layout matches the task.
